# Portfolio Website

This is my personal portfolio website created using HTML and CSS.
It is designed to present my profile, skills, and projects in a simple and professional way.

The website includes sections such as About Me, Skills, Projects, and Contact.
It helps in showcasing my work and learning progress as a BCA student and aspiring developer.

This project was also used to understand how to create a GitHub repository and deploy a website using GitHub Pages.

The website is successfully deployed and accessible online through GitHub Pages.
